Shortcodes for linking to the AI Agent Skills directory.

== Description ==

AI Agent Skills adds a simple shortcode for publishing links to AI Agent Skills,
a curated directory for Claude skills, Codex skills, and AI agent workflows.

Homepage: https://aiagentskills.net

== Installation ==

1. Upload the plugin directory to `/wp-content/plugins/`.
2. Activate the plugin.
3. Add `[ai_agent_skills_links]` to a post, page, or widget.
